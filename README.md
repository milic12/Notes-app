### To run app 

First clone project: 

```bash

git clone https://github.com/milic12/Notes-app.git

```

Client side:

in termial run:
```bash
cd .\notesapp.client\
```
```bash
npm install
```
then
```bash
 npm run dev
```

Server side:
```bash
cd .\NotesApp.Server\
```
then
```bash
dotnet run --launch-profile https
```
